# 🍎 Fruit & Vegetable Freshness Detector

## 📌 Overview

The **Fruit & Vegetable Freshness Detector** is a Deep Learning–based web application that predicts whether a fruit or vegetable is **Fresh** or **Rotten** from an uploaded image.

The model is built using **TensorFlow/Keras** and deployed using **Streamlit** for real-time prediction.

---

## 🚀 Features

* Upload fruit or vegetable images
* Detect freshness condition instantly
* Simple and interactive web interface
* Automatic AI model download

---

## 🛠️ Technologies Used

* Python
* TensorFlow & Keras
* Streamlit
* NumPy
* Pillow

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## ✅ Pros

* Easy-to-use interface
* Real-time prediction
* Automated model loading
* Demonstrates practical AI application

---

## ⚠️ Limitations

* Similar-looking fruits may be misclassified
* Limited dataset affects accuracy
* Background and lighting can influence results
* Example: Fresh pineapple predicted as tomato or rotten mango as potato

---

## 📂 Note

The trained model file is downloaded automatically during execution due to GitHub size limitations.
